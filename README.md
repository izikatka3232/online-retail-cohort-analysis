# online-retail-cohort-analysis
Анализ удержания клиентов с помощью Cohort Analysis на примере Online Retail II датасета.

##  Что это?

Этот проект показывает:
- **Сколько % клиентов возвращаются** после первого заказа
- **Когда происходит максимальное выбывание** клиентов
- **Какие действия помогут улучшить** удержание

###  Ключевые выводы:

1. **70% клиентов теряется в первый месяц** (M+0 → M+1)
2. **После 3-го месяца retention стабилизируется** на 20-25%
3. **Новые когорты показывают улучшение** - бизнес движется правильно
