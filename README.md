# Online Retail Cohort Analysis

Анализ удержания клиентов с помощью Cohort Analysis на примере Online Retail II датасета.

Этот проект демонстрирует применение Cohort Analysis — анализ поведения групп клиентов, объединённых по времени их первой покупки.

## Что это?

Проект отвечает на ключевые вопросы:

- Сколько % клиентов возвращаются в каждый месяц после первой покупки?
- Когда происходит максимальное выбывание клиентов?
- Какую ценность (LTV) генерирует каждая когорта за время жизни?
- Как изменяется качество привлекаемых клиентов?

## Ключевые выводы

### 1. Критический первый месяц — 79% потеря retention

Максимальный отток происходит именно в первый месяц. Это типично для e-commerce — большая часть первых покупателей — это "проверяющие" или случайные посетители.

| Период | Retention | Интерпретация |
|--------|-----------|---------------|
| M+0 | 100% | Базовый месяц (все покупатели) |
| M+1 | 21.2% | 79% клиентов не возвращаются |
| M+3 | 21.6% | Стабилизация |
| M+6 | 17.8% | Плато |
| M+12 | 18.2% | Долгосрочное ядро |

### 2. Низкая retention, но растущий LTV

Несмотря на 79% падение retention, оставшиеся клиенты генерируют значительную стоимость.

Оставшиеся 21% клиентов — это наиболее ценные и лояльные сегменты. Они составляют основу долгосрочной выручки.

| Период | LTV за период | Интерпретация |
|--------|----------------|---------------|
| M+0 | $508.31 | Первая покупка |
| M+1 | $104.69 | Повторные покупки |
| M+12 | $150.25 | Долгосрочная стоимость |

### 3. После месяца 3 — стабилизация

```
M+0 → M+1:    Максимальный drop (-79%)
M+1 → M+3:    Медленная стабилизация (-0.4%)
M+3 → M+12:   Плато (изменения минимальны)
```

Если клиент пережил первый месяц и вернулся, он с высокой вероятностью останется в течение 12+ месяцев.

### 4. Качество когорт со временем улучшается/ухудшается

```
Старые когорты (2010-12 — 2011-01):  M+1 = 100%
Новые когорты (2011-11 — 2011-12):   M+1 = 100%

Тренд по LTV: Новые когорты показывают сопоставимые значения
```

Бизнес привлекает примерно одинаковые по качеству когорты клиентов на протяжении периода.

## Визуализации в проекте

### 1. Retention Heatmap

Тепловая карта показывает % возвращающихся клиентов:

- Зелёный = высокая retention (20%+)
- Жёлтый = средняя (10-20%)
- Красный = низкая (<10%)

### 2. Average Retention Curve

Средняя кривая удержания по всем когортам. Четко виден drop в M+1 и плато после M+3.

### 3. LTV Heatmap

Денежное значение покупок по периодам. Показывает, где генерируется основной доход.

### 4. Average LTV Curve

Средний LTV за каждый период. Показывает, что хотя retention падает, LTV более стабильна.
