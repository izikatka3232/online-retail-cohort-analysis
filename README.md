# Cohort Analysis & Customer LTV

## Обзор проекта
В данном проекте проводится анализ пользовательского поведения в e-commerce с использованием когортного анализа и расчёта LTV (Lifetime Value).

Основная цель:
- понять, как пользователи удерживаются со временем  
- определить, когда клиенты приносят основную выручку  
- выявить точки потери прибыли  

---

## Цели
- Проанализировать удержание пользователей (retention) с помощью когортного анализа  
- Рассчитать LTV для разных когорт  
- Найти закономерности в поведении пользователей  
- Сформулировать рекомендации для бизнеса на основе данных  

---

## Данные
- 1,067,371 исходных транзакций
- 5,878 уникальных клиентов
- 4,631 уникальный товар
- Период: Dec 2009 - Dec 2011
- Источник: [Kaggle](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)

---

## Используемые технологии
- Python  
- pandas — обработка данных  
- numpy — численные вычисления  
- matplotlib / seaborn — визуализация  

---

## Этапы анализа

### 1. Очистка данных
- удалены пропущенные значения  
- отфильтрованы отрицательные значения  
- создана метрика Revenue  

### 2. Когортный анализ
Определены:
- `cohort_month` — месяц первой покупки  
- `order_month` — месяц покупки  
- `cohort_index` — номер периода  

Построена когортная таблица и рассчитан retention  

### 3. Анализ удержания
- построена матрица retention  
- визуализация в виде heatmap  

Основная идея — оценить, сколько пользователей возвращаются после первой покупки  

### 4. Расчёт LTV
- рассчитана выручка по когортам  
- посчитан средний доход на пользователя  
- проанализирована динамика LTV  

---

## Ключевые выводы
- Удержание пользователей резко снижается после первого месяца  
- Большинство пользователей не возвращаются после первой покупки  
- Основная выручка формируется в первые 2–3 месяца  
- Некоторые когорты показывают лучшие результаты, что может быть связано с внешними факторами (например, акциями)  

---

## Рекомендации
- Улучшить онбординг пользователей  
- Внедрить механики для повторных покупок (скидки, бонусы)  
- Сфокусироваться на первых 30 днях жизни пользователя  
- Проанализировать успешные когорты и масштабировать их практики  
- Использовать инструменты удержания (email-рассылки, push-уведомления)  

---

## Заключение
Анализ показывает, что удержание пользователей является ключевым фактором формирования LTV. Улучшение раннего взаимодействия с пользователем может существенно увеличить выручку.
